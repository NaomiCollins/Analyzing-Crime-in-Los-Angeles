📊 Analyzing Crime in Los Angeles

Los Angeles 😎 The City of Angels. Known for Hollywood, warm weather, and iconic culture. But like any major metropolitan city, it also faces significant crime challenges.
In this project, I analyzed crime data to uncover patterns in criminal activity and generate insights that could support smarter resource allocation decisions.

🎯 Objectives

Using the crimes.csv dataset, I answered three key questions:
⏰ Identified the hour with the highest crime frequency (peak_crime_hour)
🌙 Determined which area has the most night crimes (10PM–3:59AM) (peak_night_crime_location)
👥 Analyzed crime distribution across victim age groups (victim_ages)

🛠 Tools Used

Python • Pandas • Data Cleaning • Exploratory Data Analysis
